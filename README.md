## Purpose

Pigment.O is a Krita plugin Color Picker and Mixer.

## Features

* Harmony Colors
* Various GUI Panels
* Various Channel Sliders and Values
* Color Mixer
* Pinned Colors
* Color History
* Read ![wiki](https://github.com/EyeOdin/Pigment.O/wiki) for more information
